# joncoPS1
Custom $PS1 for shell - a lot of amazing features and languages are planned

## Currently supports
### OS
* Linux

### Shells
* bash

## Usage
### Run in bash shell
```
git clone https://github.com/JoncoLab/joncoPS1.git && \
chmod +x ./install && \
bash ./install
```
